# AGENTS.md - React-Vite Full-Stack Project

## Build/Test/Lint Commands

- `npm run build` - runs vite build
- `npm start` - runs node src/server/server.js

## Code Style Guidelines

- **Formatting**: 2-space indentation, double quotes
- **Naming**: camelCase for variables/functions
- **Database**: MongoClient with mongoose@8.16.1, schema-first approach

## Project Rules

- Keep functions single-purpose unless composable/reusable
- Avoid unnecessary destructuring and else statements
- Prefer single-word variable names where possible
